# 📈 Technical Indicator–Driven Stock Prediction Using Machine Learning

## 🚀 Project Overview

This project builds a modular stock prediction framework using OHLC market data enriched with technical indicators.  

The system integrates:

- Technical Analysis (RSI, MACD, Bollinger Bands, Moving Averages)
- Feature Engineering
- PCA (Dimensionality Reduction)
- Regression & Classification Models
- Clustering for Market Regime Detection
- Neural Networks for Price Forecasting
- Data Visualization

The objective is to predict next-day stock price and trend direction using machine learning techniques.
